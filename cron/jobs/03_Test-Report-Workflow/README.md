# 03_Test-Report-Workflow

汎用のスキルとしてWeb情報を収集してレポート化するworkflowを規定する。

## 目的
- 定刻に自動実行されるcronジョブの出力管理
- 英語原文 + 日本語逐語訳のレポートを日付ごとに保存
- 過去の調査結果の蓄積と参照

## 運用ルール
- プロンプトは `prompt.md` に最新版を保存
- 動作フローは`workflow.md`に最新版を保存
- 使用するファイル定義は`filesets.md`に最新版を保存
- レポートの定義は`report_definition.md`に最新版を保存
