# 02_Hermes-Usage-Research

Hermes Agent の活用事例・知見を note、Qiita、Zenn、個人ブログなどから収集する朝のリサーチ用フォルダ。

## 目的
- 毎日5:00に自動実行されるcronジョブの出力管理
- Hermesの効果的な使い方に関する最新情報を蓄積
- ユーザー自身がHermesをより良く活用するための参考資料

## 運用ルール
- 出力ファイル: `outputs/YYYY-MM-DD.md`
- エラーログ: `logs/` に日付付きで保存
- プロンプトは `prompt.md` に最新版を保存
- 有用な情報は `research/hermes-usage-research/` へ月次で整理
