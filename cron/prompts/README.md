# cron/prompts/

cronジョブで使用しているプロンプトのバックアップ・バージョン管理用フォルダ。

## 目的
- 各cronジョブのプロンプトを安全に保存
- プロンプト変更時の差分管理
- 新規cron作成時のテンプレートとして活用

## 運用ルール
- ファイル名は `01_Quara-English-Research.md` のようにジョブ名を付ける
- プロンプトを更新した場合はこのフォルダと各jobフォルダの `prompt.md` の両方を更新する
- 古いバージョンのプロンプトは `_v1.md` などの命名で残す
